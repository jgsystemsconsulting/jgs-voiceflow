# Security Policy

This policy covers the **VoiceFlow installer and application** distributed from this repository.

## Reporting a vulnerability

VoiceFlow is closed-source, so please **do not** open a public GitHub issue for security matters.
Instead, report privately:

- **Email:** [support@jgsystemsconsulting.com](mailto:support@jgsystemsconsulting.com)
- **Or:** a [GitHub private security advisory](https://github.com/jgsystemsconsulting/jgs-voiceflow/security/advisories/new)
  (if you prefer to report through GitHub).

Please include:

- The VoiceFlow version (tray → **About**).
- Your Windows version.
- A clear description of the issue and the security impact.
- Steps to reproduce, and any proof-of-concept, logs, or screenshots.

## What to expect

- **Acknowledgement within 5 business days.** (This is a response-time target, not a fix-time
  guarantee.)
- We will keep you informed as we investigate and, where appropriate, credit you once a fix ships.

If you need to send sensitive details, ask in your first email and we can arrange an encrypted
channel. Thank you for helping keep VoiceFlow users safe.
