§01 — Product

VoiceFlow

Push-to-talk voice dictation for Windows. Hold a hotkey, speak, release — VoiceFlow transcribes your speech on-device, cleans it into tight, ready-to-use text, and pastes it at your cursor. Local-first and private by default; free to use.

Download the installer → View the repository →

§02 — What it does

Push-to-talk

A global hotkey (default Ctrl+Space) — hold or toggle. Speak, release, done.

On-device transcription

Speech-to-text runs locally with Whisper. No audio is uploaded on the default path.

Clean text

Removes filler ("um", "uh", false starts), fixes punctuation, and can restructure rambling speech.

Paste at the cursor

Inserts the result where you are typing, clipboard-safe — your previous clipboard is restored.

Lives in the tray

A system-tray app with a mode menu, settings, and optional start-on-login.

Fully offline

The installer bundles the speech models; cloud services are opt-in only.

§04 — Install

1

Download

Get VoiceFlow-Setup.exe (~700 MB; offline models bundled) from the latest release.

2

Run it

Per-user install — no administrator rights. VoiceFlow is not code-signed yet, so Windows may warn about an unknown publisher: choose More info → Run anyway.

3

First run

VoiceFlow profiles your hardware and picks the optimal local model — automatically, on-device.

4

Dictate

Hold Ctrl+Space, speak, release. The cleaned text is pasted at your cursor.

§05 — Privacy

On the default path, no audio and no text ever leave your machine — speech-to-text runs on-device and cleanup uses a built-in offline cleaner. Cloud services are strictly opt-in, and you can verify zero network use yourself.
